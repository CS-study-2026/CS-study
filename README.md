# 2026 CS Study 📚

CS 전반을 정리하고 면접/실무에 바로 활용할 수 있는 기반을 만드는 스터디입니다.  
컴퓨터 구조 → 운영체제 → 자료구조 → 네트워크 → 데이터베이스 순으로 진행하며,
모든 정리는 노션과 깃허브에 함께 기록합니다.

---


## 📌 Ground Rule

### when 
  - 매주 **화요일** 정규 일과 후 **오후 6시**
### 기록 
  - 스터디 **노션 페이지** 정리 + **공용 GitHub 레포지토리**에 업로드
### 발표 순서 🔥
  - 혜림 진석 영관 → 재봉 서형 혜민 → 지영 이지 서지

---

## 🧾 진행 방식

### 1. 정규 스터디

- **발표자 3인**
  - 각자 맡은 주제에 대해 **노션에 내용 정리** 후 **발표**
- **나머지 6인**
  - 발표 범위 **사전 학습(훑어오기)**
- **스터디 시간 내**
  - 발표 후 **질문 시간**
  - 각자 공부한 내용 및 관련 논의 **자유 공유**
- **스터디 종료 후**
  - 발표자 3인은  
    - 기존에 노션에 정리한 내용 + 스터디 시간에 나온 추가 내용 반영  
    - 각자 담당 분량을 **깃허브 레포지토리에 업로드**

> 정규 스터디는 “교재/정석”에 가까운 CS 기본기를 다지는 시간입니다.


### 2. 자율 스터디

- **주제 예시**
  - 지난 주 메인 주제와 관련된 **최신 이슈/심화 내용**
  - 그 외 관심 있는 **IT 트렌드나 기술 이슈**
- **진행 방식**
  - 사전에 **노션에 간단 정리**
  - 스터디 후 정리본을 **깃허브에 업로드**  
    (정규 스터디만큼 길지 않아도 괜찮음)

> 자율 스터디는 각자의 관심사와 최근 트렌드를 공유하는 “확장/응용” 시간입니다.
